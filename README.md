# JoeLang-documentation
Documentation for my future penguinmod project

# Syntax

#### 1 - Every command must start must start with joe. (this is just for style purposes, might change in a future

#### 2 - Every command must have squared parenthesis [ and ] this helps the system check if the command actually exists

(if it dosent have them it is very probable that you will get a "syntax error, command does not exist")

#### 3 - The command name always goes after joe. and before the first [

#### 4 - Tags go at the start and end of the command you want to place it

#### 5 - Reporters usually go inside strings or number in commands, there are a few exceptions tho

#### 6 - Every command should be writen in lowercases (if you write in uppercase it will transform it to lowercase anyways)

#### 7 - In tags squared parenthesis go at the start and end of the command name

#### 8 - Tags and reporters do not need joe. at their start

# Arguments

#### 1 - "" for strings

#### 2 - '' for numbers

#### 3 - || for vars 
(as i write this you cant make vars yet, the only accsesible var is the answer one as it is made by default)

# Commands
#### 1 - joe.says["insert text here"] - Prints text into the output

#### 2 - joe.what["insert question here"] - Works like an input from js or python

#### 3 - joe.sleepy['1'] - Waits the seconds set in the input

#### 4 - joe.clear[] - clears the output

# Tags

#### 1 - [loop] - loops the command the tag is in

# Reporters

#### 1 - math.['1 + 1'] - runs any equation no matter what operator uses or the length of the operator
